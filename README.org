* Introduction

~resworb~ is a Python library for manipulating browser data.

/Currently on tested on macOS./

** Supported resources (WIP)

|              | Safari | Firefox | Chrome |
|--------------+--------+---------+--------|
| Opened tabs  | ✅     | ✅      |        |
| Cloud tabs   | ✅     |         |        |
| Reading list | ✅     |         |        |
| Bookmarks    | ✅     | ✅      | ✅     |
| Histories    | ✅     | ✅      | ✅     |
